# Figure style

All generated figures shown in this documentation site use a shared plotting style implemented in `src/nslab2d/plot_style.py`.

The style was chosen to follow common academic-figure guidance:

- standard sans-serif fonts
- colorblind-safe categorical colors
- perceptually sensible colormaps
- minimal decoration
- vector-friendly export settings

For the detailed discussion, see the original page:

- [Unified figure style](../figure_style.md)
